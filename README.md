# Sam Meske Portfolio

Personal portfolio site for Sam Meske (AI product strategy + systems engineering).

## Development

```bash
cd vision-venture-site
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
cd vision-venture-site
npm run build
```

## Domain

The canonical site URL is set via `NEXT_PUBLIC_SITE_URL` (defaults to `https://sammeske.com`).
